Intro jQuery

Why does it exist?

Three reasons: It just gives us "syntactical sugar". It used to be about cross browser compatibility but that is largely a non issue now (so really just one reason) It's so popular that it has a virality effect (we teach b/c it's so popular)

How much additional behavior does it add to the browser?

none; it works within the constraints of what the browser API can do for us.

Library or framework? Why?

Library, explain the difference (keep in mind they haven't seen a framework yet most likely).

Expectations of You

The jQuery docs are quite good, you should be using them a lot The API has alternative ways of doing things (eg .on('click') vs click()) and you should not get hung up on learning the entire surface area for jQuery. The goal is to learn it enough and get comfortable with the process of learning it (yes, this is a recurring theme!)
